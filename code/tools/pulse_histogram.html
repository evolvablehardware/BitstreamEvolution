pulse_histogram.py¶

Pulse Histogram¶

This program that takes an experiment, looks at all pulse counts, and generates a histogram. This was used to see the number of time-outs vs. number of 0s vs. number of times close to target.

This program is ment purely to assist with analysis or finding and explaining errors.
